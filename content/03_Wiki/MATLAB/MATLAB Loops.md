# For-Loop

> [!NOTE] Syntax
> ```matlab
> for index = values
> statements
> end
> ```

> [!EXAMPLE]
> ```matlab
> for i = 1:5:10
> 	i
> end
> 
> % Output:
> % i = 1
> % i = 6
> ```

> [!EXAMPLE] For specified values
> ```matlab
> for v = [1 5 8 17]
> 	disp(v)
> end
> 
> % Output:
> % 1
> % 5
> % 8
> % 17
> ```

# While
> [!NOTE] Syntax
> ```matlab
> while (statement)
> 	statements
> end
> ```
