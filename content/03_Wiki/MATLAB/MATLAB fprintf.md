`%s` - print a string
`%c` - print a single character
`%d` or `%i`- print a whole number
`%f` - print a floating point number
`\n` - print a new line (go to the next line to continue printing)
`\t` - print a tab
`\\` - print a slash
`%%` - print a percent sign
