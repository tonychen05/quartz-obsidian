
> [!INFO] Even Functions
> An even function is symmetric about the y-axis. In other words, the function is even if: $$
 f(-x)=f(x), \text{ for all }x
$$

> [!INFO] Odd Functions
> An odd function is symmetric about the origin. In other words, the function is odd if:
> $$
 f(-x)=-f(x), \text{ for all } x
$$
